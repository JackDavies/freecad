\section{Chair dice jail}
	\subsection{Parts needed}
		x2 chair-side, x2 chair-brace, x1 chair-back and x1 chair-seat

	\subsection{Build steps}
		Step 1. Glue chair-back and 2 braces into the first chair-side part		
		
		\includegraphics[width=0.4\linewidth]{img/jpg/chair-1.jpg}		
		
		Step 2. Insert the chair-seat into chair-side, \textbf{don't} glue as this needs to be free to move
				
		\includegraphics[width=0.4\linewidth]{img/jpg/chair-2.jpg}		
		\newpage
		Step 3. Glue the second chair-side part to the braces and chair back, make sure the chair seat is free to move 

		\includegraphics[width=0.4\linewidth]{img/jpg/chair-3.jpg}	

\newpage		
\section{Table rolling tray}
	\subsection{Parts needed}
		x1 table-base, x2 table-leg-side, x2 table-side, x2 table-insert, x1 table-top-frame, optional x4 table-topper

	\subsection{Build Guide}
		Step1. Glue 2 table-leg-side to either side of the table base 
		
		\includegraphics[width=0.5\linewidth]{img/jpg/table-1.jpg}

		Step2. Glue 2 table sides to the table base and the table leg sides, make sure the joints are flush

		\includegraphics[width=0.5\linewidth]{img/jpg/table-2.jpg}		
		
		\newpage
		
		Step3. Glue 2 table-inserts to the table base along the inside face of each of the table-leg-side pieces. Make sure the insert is flat along the table base.

		\includegraphics[width=0.5\linewidth]{img/jpg/table-3.jpg}	
		
		Step4. Glue table-top-frame to the top faces of the table-leg-side and table-side pieces. Don't cover the table-insert pieces with the top frame

		\includegraphics[width=0.5\linewidth]{img/jpg/table-4.jpg}	
		
		\newpage
				
		Step5. As an optional step you can print 4 table-topper pieces that can be placed within the table-top-frame

		\includegraphics[width=0.5\linewidth]{img/jpg/table-5.jpg}
		
		Table with 4 toppers
